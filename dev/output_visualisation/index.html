Output visualisation

The auxiliary package ConsensusBasedXPlots.jl provides routines to visualise the output of minimise for problems in one or two dimensions. These routines are kept as a separate package in order to minimise the dependencies of ConsensusBasedX.jl, since not every user will require visualisation tools.

To plot the output of CBO, simply run minimise with extended_output = true (see Extended output), and then call the plot_CBO method:

out = minimise(f, D = 2, extended_output = true)
using ConsensusBasedXPlots
plot_CBO(out)
